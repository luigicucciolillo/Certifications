# Introduction to DevOps and Cloud Infrastructure Technologies

A series of 4 courses hold by The Linux Foundation followed to get a strong base and understanding of the Cloud Environment and the DevOps mentality and methodologies.

Teching material for (estimated) 112 hours of courses selfpaced.

---

## [Introduction to Jenkins (LFS167) - 20 hours](https://github.com/luigicucciolillo/Certifications/tree/main/The%20linux%20foundation/Introduction%20to%20Jenkins%20LFS167)
 - Studied the fundamentals of Jenkins as a CI/CD automation tool within modern DevOps workflows. Explored Jenkins installation and configuration, plugin management, and essential security practices. Worked with Freestyle and Pipeline project types to understand build automation. Gained hands-on familiarity with Jenkinsâ€™ role in continuous integration and delivery, including its extensibility, distributed build capabilities, and scalability features.

---

## [Introduction to kubernetes (LFS158) 20 hours](https://github.com/luigicucciolillo/Certifications/tree/main/The%20linux%20foundation/Introduction%20to%20Kubernetes%20LFS158)
 - Completed an introductory course on Kubernetes, focusing on control-plane components, cluster lifecycle operations, and container orchestration. Provisioned and administered a Minikube cluster and worked with core primitives such as Pods, ReplicaSets, Deployments, Services, Ingress, Volumes/PVCs, ConfigMaps, and Secrets. Studied RBAC, TLS-based authentication, authorization modes, and admission controllers. Applied best practices for deploying, scaling, and updating containerized microservices in cloud-native environments.

## [Introduction to Cloud Infrastructure Technologies (LFS151) 50 hours](https://github.com/luigicucciolillo/Certifications/tree/main/The%20linux%20foundation/Introduction%20to%20Cloud%20Infrastructure%20Technologies%20LFS151)
 - Introductory course on cloud infrastructure technologies, covering virtualization, IaaS/PaaS/CaaS models, Docker containerization, microservices, unikernels, and container orchestration. Studied SDN, SDS, and automation tooling for configuration management, image building, and debugging/monitoring of containerized workloads. Gained practical exposure to cloud-native delivery patterns including serverless computing, service meshes, IoT integration, and distributed tracing, building a solid foundation for evaluating and implementing open-source, scalable infrastructure solutions.
